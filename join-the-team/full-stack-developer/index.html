Full-Stack Developer AKA Software 'BadAss' Engineer

Somos o time de desenvolvimento do maior e-commerce de moda da América Latina, a Dafiti. Aqui mantemos e evoluimos os sistemas da Dafiti, Kanui e Tricae.

Se você é criativo e com bom senso, é flexível e comprometido, tem paixão por tecnologia e engenharia de alta qualidade, venha trabalhar conosco, pois temos muito em comum :)

Sobre o trabalho

Como um desenvolvedor full-stack você fará parte de um time de desenvolvedores muito habilidosos, designers e product owners. Palavras como Dependency Injection, Domain Driven Design, Hexagonal Architeture, CQRS, Isomorphic JavaScript e Test Driven Design não são totalmente estranhas para você, mesmo que não domine totalmente suas técnicas.

Acima de tudo, gostamos de criar valor para o nosso cliente, enquanto garantimos um sistema de alta performance, alta disponibilidade, robusto e resiliente para escalar com nosso tráfego.

Boa parte do trabalho é feita no back-end, no entanto você deve se sentir confortável em interagir com o front-end, em resumo: esperamos que você se envolva em todas as partes do desenvolvimento do produto.

No dia-a-dia você irá ajudar no refinamento de requisitos, planejamento e execução das tarefas que serão realizadas pelo time, POCs relacionadas à evolução do produto e elaboração de techtalks, além de discutir bastante sobre arquitura para sistemas de larga escala.

Nossos sistemas estão mudando de um sistema monolítico para serviços e micro-serviços onde utilizaremos a melhor ferramenta para cada caso. Atualmente temos sistemas en PHP, Python, Java, JavaScript e GO.

Qualificações
  • Superior completo em Ciencias da Computação ou áreas correlatas.
  • Experiencia em desenvolvimento de software em uma ou mais linguagens de alto nível.
  • Experiencia em dois ou mais: web application development, Unix/Linux environments, mobile application development, distributed and parallel systems, machine learning, developing large software systems, NoSQL/SQL databases and/or security software development.
